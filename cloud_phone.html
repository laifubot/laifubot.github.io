                # 云手机上脚本使用方法

                云手机是在云端服务器上模拟的一个安卓手机环境，可以在上面挂机手游而不占用手机的资源，云手机客户端可以将云手机的画面传输给自己的手机上而监控云手机的运行状况
                欢迎加入QQ交流群799465338，有些游戏问题和脚本使用问题可以在交流中询问解决

                TIP

                云手机是收费的

                # 下载购买云手机

                目前市场上有很多云手机品牌可以自行选择，这里推荐多多云和红手指：
                多多云云手机 (opens new window)
                红手指云手机 (opens new window)

                # 将软件安装到云手机中

                云手机一般都支持将自己手机中的软件上传到云手机中，也可以直接在云手机中下载安装
                1、将游戏安装到云手机中 2、将加速器安装到云手机中 3、点击下载脚本 (opens new window)，将脚本安装到云手机中

                # 登录游戏

                如果您是第一次在云手机中登录游戏，则需要同步自己的游戏账号到虚拟机中：
                1、打开加速器开启加速
                2、点击运行游戏王决斗链接
                3、在游戏登录页面中点击数据转移

                4、再点击从其他设备转移数据，然后登录自己的游戏账号就可以了

                # 运行脚本

                点击模拟器中的来福bot图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
                点击悬浮按钮后会显示运行按钮，点击运行即可显示脚本配置UI页面，设置自己要执行的任务，勾选自己的卡组，然后点击开始按钮即可开始
