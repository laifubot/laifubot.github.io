# 国际服手机端使用说明

手机上使用脚本，目前只支持在安卓手机上使用，苹果手机用户可以将账号迁移到安卓手机上使用，或者在电脑上使用
手机上使用需要在手机上下载虚拟机，可以实现后台挂机
欢迎加入QQ交流群799465338，有些游戏问题和脚本使用问题可以在交流中询问解决

# 在手机上下载虚拟机

这里提供了免广告版本的X8沙箱虚拟机： 点击跳转下载虚拟机 (opens new window) 下载完之后在手机上安装即可
也可以使用其他虚拟机，如果这个虚拟机比较卡，群文件还有一个51虚拟机，如果打开虚拟机时提示需要激活可以下载启动器修复激活一下

TIP

这个虚拟机是免费的，如果弹出购买VIP页面，关闭掉就可以，使用其他虚拟机也可以

# 设置虚拟机

虚拟机下载安装完之后，打开虚拟机，在虚拟机的设置中修改分辨率为720*1280 320dpi

# 在手机中安装加速器

也可以在虚拟机中安装加速器，但是可能会有些麻烦，推荐一个免费的加速器：点击下载猎豹加速器 (opens new window)，直接在手机上安装就可以，
在手机上虚拟机中使用的加速器推荐使用ourplay加速，点击下载ourplay (opens new window) ,使用ourplay加速注意要打开ourplay中的游戏，脚本设置要勾选国际服(谷歌空间)
也可以使用其他加速器

# 在虚拟机中安装游戏和脚本

1、如果您手机上已经安装了游戏，直接中虚拟机中导入即可，如果手机上还没有下载游戏，可以在手机上安装游戏后导入虚拟机，也可以直接在虚拟机中下载安装
如果导入游戏后打开游戏显示下面这个错误：
则重启虚拟机即可
2、点击下载脚本 (opens new window)，可以安装到手机上后导入虚拟机，也可以直接中虚拟机中打开下载链接下载安装

TIP

第一次安装脚本后如果打不开，则重启虚拟机即可

# 登录游戏

如果您是第一次在虚拟机中登录游戏，则需要同步自己的游戏账号到虚拟机中:
如果是使用ourplay加速，则需要在ourplay中打开游戏转移数据
1、打开加速器开启加速
2、点击运行游戏王决斗链接
3、在游戏登录页面中点击数据转移

4、再点击从其他设备转移数据，然后登录自己的游戏账号就可以了

TIP

如果使用ourplay转移数据，确认正确的账号密码，却一直显示密码错误，则切换ourplay的加速节点在试一下，如果所有加速节点都不行，则下载猎豹加速器 点击下载猎豹加速器 (opens new window)，在虚拟机外面安装并打开猎豹加速器加速，然后在虚拟机里转移数据

# 运行脚本

点击模拟器中的来福bot图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
点击悬浮按钮后会显示运行按钮，点击运行即可显示脚本配置UI页面，设置自己要执行的任务，勾选自己的卡组，然后点击开始按钮即可开始
