# 国服电脑模拟器端使用说明

  • 国服查的比较严，请谨慎使用，国服会检测手机上的虚拟机，所以目前国服只推荐中电脑模拟器中使用
  • 电脑模拟器，指的是在PC电脑上的安卓模拟器中使用，安卓模拟器在电脑中模拟了一个安卓手机系统环境，可以运行安卓应用程序，此种使用方法需要在电脑上设置

注意

不要在模拟器中安装其他脚本或者其他游戏的脚本，可能会被检测封号
国服游戏不要在手机虚拟机或者云手机中打开，会被封号

# 下载安装安卓模拟器

如果您还没有在电脑上安装安卓模拟器，可以点击下载雷电安卓模拟器 (opens new window)
只推荐使用雷电模拟器，点链接就可以下载4.0版本，不要下载雷电9

# 修改模拟器设置

模拟器安装完之后打开，将模拟器分辨率修改为 手机版720*1280， 确认开启了root权限
修改设置之后重启一下模拟器

# 下载安装游戏到模拟器中

点击下载国服网易官方游戏王决斗链接 (opens new window)
点击下载国服B站渠道游戏王决斗链接 (opens new window) 下载完成后将安装包拖拽进模拟器中安装

# 下载脚本到模拟器中

点击下载脚本 (opens new window)，下载完成后，将下载的脚本安装包拖拽进模拟器中即可完成安装

# 运行脚本

点击模拟器中的来福bot图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
点击悬浮按钮后会显示运行按钮，点击运行即可显示脚本配置UI页面，设置自己要执行的任务，勾选自己的卡组，然后点击开始按钮即可开始
