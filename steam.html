# 国际服steam端使用说明

steam端，指的是在电脑上的steam平台上下载的游戏王决斗链接客户端，需要在电脑上使用
欢迎加入QQ交流群799465338，有些游戏问题和脚本使用问题可以在交流中询问解决

# 下载游戏王决斗链接steam版本客户端

如果您已经下载了游戏王决斗链接steam版客户端，可以跳过这一步骤
由于游戏王决斗链接国际服并没有在中国区上架，所以中国大陆地区的steam账号在steam上是搜不到游戏王的，您可以注册或者淘宝上购买一个非中国大陆地区的steam账号来操作，自己注册的中国区steam账号也可以用一些特殊方法来入库下载游戏王点击查看下载入库方法 (opens new window)
您也可以百度或者b站搜索最新的游戏王steam下载入库方法

# 下载脚本

如果您还没有下载脚本点击下载steam端脚本 (opens new window)
下载之后直接安装就可以，如果安装时显示解压错误可以进群下载压缩包版本，解压后可以直接使用，如果安装出现其他问题可以联系淘宝客服或者群主解决
安装完成后，桌面上会显示来福bot图标，点击即可运行脚本

# 游戏内设置

1、进入游戏后确认游戏的语言是简体中文
2、尽量将卡垫和卡套修改成默认的，否则可能导致覆盖的魔法卡无法发动
3、游戏中决斗设置，设置镜头顶部视角，关闭3D效果和角色台词

# 运行脚本

TIP

本运行期间游戏窗口可以被遮挡不影响正常使用电脑，但是不可以最小化游戏窗口

运行游戏脚本软件，设置要执行的任务，点击开始按钮开始运行，任务列表中添加了定时清理路人后会默认30分钟清理一次路人，其他任务会按顺序执行，执行完一个任务后执行下一个任务

任务设置页面可以设置每个任务执行的参数，比如可以设置清理路人的间隔时间（时间单位都是分钟）、使用的卡组（卡组1、2、3是角色所拥有的卡组从左往右数依次为1、2、3组）、传送门的等级和倍数等，这里设置的执行时长（单位分钟）时间到了之后这个任务执行完会切换到下一个任务
