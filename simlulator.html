                      # 国际服电脑模拟器端使用说明

                      电脑模拟器，指的是在PC电脑上的安卓模拟器中使用，安卓模拟器在电脑中模拟了一个安卓手机系统环境，可以运行安卓应用程序，此种使用方法需要在电脑上设置
                      欢迎加入QQ交流群799465338，有些游戏问题和脚本使用问题可以在交流中询问解决

                      # 下载安装安卓模拟器

                      如果您还没有在电脑上安装安卓模拟器，可以点击下载雷电安卓模拟器 (opens new window)
                      也可以使用其他安卓模拟器

                      # 修改模拟器设置

                      模拟器安装完之后打开，将模拟器分辨率修改为 手机版720*1280， 确认开启了root权限
                      修改设置之后重启一下模拟器

                      # 下载安装游戏到模拟器中

                      可以在模拟器中应用商店中搜索"游戏王决斗链接"，注意要是国际服的:
                      点击下载即可下载到模拟器中，如果您的模拟器应用商店中搜索不到游戏王决斗链接国际服的，可以先中模拟器中下载个腾讯加速器，然后在加速器中搜索下载游戏

                      # 下载安装加速器到模拟器中

                      游戏王决斗链接国际服版本由于服务器在海外，所以要登录游戏需要使用加速器，可以使用免费的“腾讯加速器”，也可以使用其他加速器，加速器下载同样可以在模拟器的应用商店中搜索下载，也可以通过其他渠道下载，例如可以百度搜索相关加速器去官网下载
                      这里给一个腾讯加速器的下载地址：点击下载腾讯加速器 (opens new window)

                      TIP

                      加速器要下载手机版本的，安装包文件后缀为.apk，下载到电脑上之后将安装包拖拽到模拟器中即可安装

                      # 下载脚本到模拟器中

                      点击下载脚本 (opens new window)，下载完成后，将下载的脚本安装包拖拽进模拟器中即可完成安装

                      # 登录游戏

                      如果您是第一次在模拟器中登录游戏，则需要同步自己的游戏账号到模拟器中：
                      1、打开加速器开启加速
                      2、点击运行游戏王决斗链接
                      3、在游戏登录页面中点击数据转移

                      4、再点击从其他设备转移数据，然后登录自己的游戏账号就可以了

                      # 运行脚本

                      点击模拟器中的来福bot图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
                      点击悬浮按钮后会显示运行按钮，点击运行即可显示脚本配置UI页面，设置自己要执行的任务，勾选自己的卡组，然后点击开始按钮即可开始
